# After-Dark
After Dark neovim colorscheme based on the after-dark VSCode colorscheme :)

## Installation
```lua
return {
  "Kopihue/after-dark",
  config = function()
    require("after-dark").setup({})
  end
}
```

## Configuration
```lua
require("after-dark").setup({
  transparency = false,
  bold = true,
  italic = true,
})
```
